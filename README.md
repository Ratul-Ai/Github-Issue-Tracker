## 1️⃣ What is the difference between var, let, and const?
#### ✏️ var,let and const they all are javascript variables.The difference between them is in scope,reassignment and redeclaration.var is a **function scope** and it can be **redeclare** and it's value can be **reassign**. Where const and let are **block scope** means it can be use other than it's assign block.And they both can't be **redeclare** in the same scope.The difference between let and const is the value of let can ber **reassign** but the value of const can't be **reassign**.It's value should be assign at declaration.

## 2️⃣ What is the spread operator (...)?

#### ✏️ The spread operator (...) is used to expand elements of an array or properties of an object into **individual elements**.It is commonly used for copying, merging, or passing values.

## 3️⃣ What is the difference between map(), filter(), and forEach()?

#### ✏️ map(),filter(), and forEach() they all are array methods.The difference between them is in **return values** and **functionality**. map() and filter() returns a new array,the difference is in functionality.map() do some work with each element and return a new array where filter() filter elements by any condition and return the filtered array.On the other hand forEach() dose not return any value but do some functionality with each elements of the array.

## 4️⃣ What is an arrow function?

#### ✏️ An arrow function is a shorter syntax for writing functions.It uses **'=>'** symbol to express a function. It can't be directly called but can be called when stored in a variable.It is mostly used for short called back functions.

## 5️⃣ What are template literals?

#### ✏️ Template literals are a way to make dynamic strings with **backticks ``**. They allow to embed variables and expressions inside strings using **${}** and allow multiline strings.
