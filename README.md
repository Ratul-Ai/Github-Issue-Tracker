## 1️⃣ What is the difference between var, let, and const?
#### ✏️ var, let, and const are all JavaScript variables.The difference between them is in scope,reassignment and redeclaration.var has a **function scope** and it can be **redeclared** and its value can be **reassigned**. Where let and const have **block scope**, which means they can only be accessed inside the block {} where they are declared.And they both can't be **redeclared** in the same scope.The difference between let and const is the value of let can be **reassigned** but the value of const can't be **reassigned**.It's value should be assigned at declaration.

## 2️⃣ What is the spread operator (...)?

#### ✏️ The spread operator (...) is used to expand elements of an array or properties of an object into **individual elements**.It is commonly used for copying, merging, or passing values.

## 3️⃣ What is the difference between map(), filter(), and forEach()?

#### ✏️ map(),filter(), and forEach() are all array methods.The difference between them is in **return values** and **functionality**. map() and filter() return a new array,the difference is in functionality.map() does some functionality with each element and return a new array where filter() filters elements by any condition and return the filtered array.On the other hand forEach() dose not return any value but do some functionality with each element of the array.

## 4️⃣ What is an arrow function?

#### ✏️ An arrow function is a shorter syntax for writing functions.It uses the **'=>'** symbol to express a function.It is mostly used for short callback functions.

## 5️⃣ What are template literals?

#### ✏️ Template literals are a way to make dynamic strings with **backticks ``**.They allow embedding variables and expressions inside strings using **${}** and allow multiline strings.
